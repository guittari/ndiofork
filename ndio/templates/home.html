BossDB BICCN Metadata

This application hosts BICCN-compatible metadata for the BossDB database.

The root of this versioned API is available at https://metadata.bossdb.org/api/v1/. Each endpoint response will include a key titled endpoints which can be used to navigate up and down the API hierarchy. You can also reach an API site map in the response of the root endpoint under the key endpoints.

To get started, try a sample of the code below.

For more information, or to report an issue, please contact metadata@bossdb.org.

Sample Code

List experiments

Python 
import requests

result = requests.get("https://metadata.bossdb.org/api/v1/experiments").json()
print(result)
JavaScript 
fetch("https://metadata.bossdb.org/api/v1/experiments")
    .then(response => response.json())
    .then(result => console.log(result));
cURL 
curl https://metadata.bossdb.org/api/v1/experiments

Get a specific experiment

Python 
import requests

experiment_id = "XXX"
result = requests.get(f"https://metadata.bossdb.org/api/v1/experiments/{experiment_id}").json()
print(result)
JavaScript 
const experiment_id = "XXX";

fetch(`https://metadata.bossdb.org/api/v1/experiments/${experiment_id}`)
    .then(response => response.json())
    .then(result => console.log(result));
cURL 
experiment_id="XXX"
curl https://metadata.bossdb.org/api/v1/experiments/${experiment_id}

Query the list of channels

Python 
import requests

query = {
    "ChannelType": "Image"
}

result = requests.post("https://metadata.bossdb.org/api/v1/channels", json=query).json()
print(result)
JavaScript 
const query = {
    "ChannelType": "Image"
};

fetch("https://metadata.bossdb.org/api/v1/channels", {
    method: "POST",
    headers: {
        "Content-Type": "application/json"
    },
    body: JSON.stringify(query)
})
    .then(response => response.json())
    .then(result => console.log(result));
cURL 
query='{"ChannelType": "Image"}'
curl -X POST -H "Content-Type: application/json" -d "${query}" https://metadata.bossdb.org/api/v1/channels
